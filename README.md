# Telegram-бот с поддержкой контекста диалога на основе локальной языковой модели. 

Репозиторий содержит скрипт для запуска телеграм бота. 

Также нужно предварительно создать Telegram бота и подключить к проекту.
Создать своего Telegram бота можно с помощью TG бота @BotFather.

## Основной функционал:
* поддержка контекста - бот помнит историю диалога для каждого пользователя
*	команда `/start` (выводится приветствие и список доступных команд)
* команда `/model` (выводит название используемой LLM)
*	очистка контекста - команда `/clear` полностью удаляет историю
* запросы пользователя пересылаются LLM, запущеной на этом компьютере, и потом ответ пересылаются пользователю
* бот многопользовательский — отдельный контекст для каждого пользователя Telegram
## Инструкция по запуску
* Клонируйте репозиторий:
  - ```bash
	 git clone https://github.com/davasilenka/NLP_lab5_bot.git
	 cd NLP_lab5_bot
* Установка зависимосте
  - ```bash
    pip install -r requirements.txt
* Для запуска необходимо предварительно установить <a href="https://lmstudio.ai/">LM Studio</a> и 
загрузить с платформы <a href="https://huggingface.co/">Hugging Face</a> языковую модель (рекомендуется: `Qwen2.5-1.5B-Instruct-Q4_K_M`).
* Запустить LM Studio, выбрать загруженную языковую модель и во вкладке Developer запустить сервер
* Создайте бота через @BotFather и получите API-токен
* В файле main.py замените значение API_TOKEN:
  - ```python
         API_TOKEN = 'ВАШ_ТОКЕН_ЗДЕСЬ'  # Замените на ваш токен
* Запустите бота
  - ```bash
    python main.py
